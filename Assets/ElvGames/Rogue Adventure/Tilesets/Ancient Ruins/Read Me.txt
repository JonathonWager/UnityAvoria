How to set up the animated tiles!

On the Animations folder you will find every animated tile to add to your Palette.
Each animations tile has a different tile size.
If you decide to create your own animations with them, here is some frame size help.

Animations01 > These are better used as 16x32 pixels per frame.
Animations02 > These are better used as 16x32 pixels per frame.
Animations03 > These are better used as 32x32 pixels per frame.
Animations04 > These are better used as 48x64 pixels per frame.

RA_Ancient_Doors can be used to create doors, they're 16x16, 32x16 and 32x32 per frame.
